# Tetris
      
### 1. 프로젝트 소개   
본 프로젝트에서는 고전 퍼즐 게임 테트리스를 이클립스를 활용하여 자바로 구현해보려고 한다.

이번 프로젝트는 혼자서 할 수 있는 SOLO mode, AI mode와 소켓 통신을 활용한 두명의 사용자가 대전을 할 수 있도록 구현하는 것을   
목표로 제작되었고, 추후 사용자 등록기능, 동시접속자 수용 등을 추가할 수 있도록 고려하고 작업하였다.

### 2. 사용된 프로그램
> JDK 1.8   
> Eclipse
